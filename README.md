# as3.ro

Static website for **as3.ro**, hosted free on GitHub Pages.

## Editing

- `index.html` — page content
- `style.css` — styling
- `CNAME` — custom domain (do not delete; it tells GitHub Pages to serve `as3.ro`)

## How it deploys

Pushing to the `main` branch publishes automatically via GitHub Pages.
Live at https://as3.ro
